# Minut take-home assignment (fullstack 2026)

## Quickstart

To get this service running locally, you need Node.js, npm and Docker installed. Then run these commands:

```bash
npm install
docker compose up --build -d
curl localhost:5006 # Should print Hello World!
```

For local development without Docker:

```bash
npm install
npm run dev
```

You're free to modify the project setup however you like, if you have a preferred way of working.
